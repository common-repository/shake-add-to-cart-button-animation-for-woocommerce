== Description ==

**Add to Cart Button Animation** is a lightweight WooCommerce plugin that adds a custom shake animation to the "Add to Cart" button on product pages. Enhance your store's user experience by drawing attention to the purchase action with this simple and effective animation.

**Free Features:**

* Enable/Disable the animation.

**Pro Version Features:**

* Fully customize animation speed and delay.
* Option to enable/disable box shadow effect.
* Customize the button's background color and text color.
* Additional animation styles and effects.
* Priority support and regular updates.

**[Upgrade to the Pro version](https://nishatsharma.net/product/shake-add-to-cart-button-animation-for-woocommerce-pro/)** to unlock these advanced features and take full control of your store's Add to Cart button animations.

This plugin is ideal for WooCommerce store owners who want to add a visual cue to encourage users to complete their purchases.

== Installation ==

1. Upload the `add-to-cart-button-animation` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to `Settings > SATCBA Button Animation` to configure the plugin settings.

== Frequently Asked Questions ==

= Does this plugin work with any WooCommerce theme? =

Yes, the plugin is designed to work with any WooCommerce theme. However, if you encounter any issues, it may be related to theme-specific CSS that conflicts with the plugin's styles.

= Can I pause the animation on hover? =

Yes, the animation will pause when the user hovers over the "Add to Cart" button.

= Is the plugin compatible with page builders like Elementor? =

The plugin is compatible with most page builders, including Elementor, as long as the "Add to Cart" button is rendered on the product page.

= How do I upgrade to the Pro version? =

You can upgrade to the Pro version by visiting [this link](https://nishatsharma.net/product/shake-add-to-cart-button-animation-for-woocommerce-pro/). After upgrading, additional features will be unlocked in the settings panel.
